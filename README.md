# SQL Server para Análise de Dados

Este repositório contém exercícios, práticas e exemplos desenvolvidos durante meus estudos em SQL Server com foco em análise de dados. O objetivo é consolidar conceitos fundamentais e avançados da linguagem SQL aplicados em cenários reais de análise e manipulação de dados.

## Conteúdos abordados

Durante o desenvolvimento deste repositório, foram trabalhados os seguintes tópicos:

- Estrutura de banco de dados relacional  
- Criação e manipulação de tabelas (DDL)  
- Inserção, atualização e exclusão de dados (DML)  
- Consultas com SELECT  
- Filtros com WHERE  
- Ordenação com ORDER BY  
- Agrupamento com GROUP BY  
- Funções de agregação (COUNT, SUM, AVG, MIN, MAX)  
- JOINS (INNER JOIN, LEFT JOIN, RIGHT JOIN)  
- Subqueries (subconsultas)  
- Tabelas temporárias  
- Views  
- Funções de data (GETDATE, DATEADD, DATEDIFF)  
- Funções de texto  
- CASE WHEN para criação de regras condicionais  
- Window Functions (RANK, DENSE_RANK, ROW_NUMBER)  
- Transações (BEGIN TRAN, COMMIT, ROLLBACK)  
- Manipulação e transformação de dados  
- Criação de relatórios com SQL  
- Integração com análise de dados e Business Intelligence  

## Objetivo

Este repositório tem como finalidade:

- Praticar consultas SQL voltadas para análise de dados  
- Consolidar conceitos de banco de dados relacional  
- Desenvolver raciocínio analítico com SQL  
- Criar exemplos reutilizáveis para projetos futuros  
- Servir como portfólio de aprendizado na área de dados  

## Tecnologias utilizadas

- SQL Server  
- SQL Server Management Studio (SSMS)  
- Banco de dados AdventureWorks  

## Estrutura do repositório

Os scripts estão organizados por temas, como:

- Consultas básicas  
- Funções de agregação  
- Joins  
- Subqueries  
- Funções de data  
- Window Functions  
- Tabelas temporárias

Cada arquivo contém exemplos comentados explicando o funcionamento dos comandos.
